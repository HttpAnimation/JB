Jailbreak

How it Works

This jailbreak works by creating a new partition table on the PC that allows any other OS with secure boot support to load. Compare the partition table on a normal PC with the one on the jailbroken PC.

Types of Jailbreaks

  1. Debian
  2. Ubuntu
  3. Windows 10/11

Debian

Debian is the recommended jailbreak and is thoroughly tested. Most things work well on this distro, but there are a few exceptions, such as Roblox, which may have performance issues.

Ubuntu

Ubuntu is the second recommended distro. While most things work, there are some minor issues. Applying a fake theme to make it look like you are not jailbroken is possible but may be a bit annoying.

Windows 10/11

Windows 10/11 is not recommended due to various issues, including slow performance and driver-related problems. Some features may not work until drivers are installed from another PC, and no support will be provided for such issues.

Booting

The way you boot the PC stays the same mainly, but there is a new menu that will show up before you can boot up the PC. This is called the Grub Boot Menu, which might look like this:

  1. Debian GNU/Linux
  2. Advanced options for Debian GNU/Linux
  3. Windows Boot Manager (on /dev/nvme0n1p1)

If you are using a Windows jailbreak or a different one, you may not see this menu.

Controlling the Boot Menu

To control the boot menu, use the arrow keys and the enter button to select and boot the desired option.
